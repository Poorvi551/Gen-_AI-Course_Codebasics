### AI is classified as 
 
 1. Generative AI
 2. Non Generative AI

1. Generative AI - AI which is used to generate or create things.

   Statistical ML -> Neural Networks -> Recurrent Neural networks -> Trasformers

   Predicting the next set of word that might come next (ex: while writing email we get a auto compleion of sentence) -> language model.

* **Language model** is an AI model that can predict the next word (or set of words) for a given sequence of words.

* Training -> self supervised learning (using fill in the blanks)
